билайн

билайн — один из крупнейших телеком-операторов в России с абонентской базой около 47 млн человек и штатом около 30 тыс. сотрудников.

Это технологичная компания изменений, чутко реагирующая на потребности рынка и людей. Оператор активно использует ИИ-технологии и Big Data как для решения задач собственного бизнеса, так и для социально-значимых проектов.

Сегодня в билайне работают более 3 тыс. IT-специалистов: 230 продуктовых команд создают более 1000 продуктов. Основной стек — Java 17, C#≥10, Python 3.9, GoLang, Kafka, Django, Spring. В портфолио продуктовых команд билайна есть решения для поиска пропавших людей с помощью Beeline AI, точной диагностики заболеваний на основе AI, распознавания речи, геоаналитики и др.
